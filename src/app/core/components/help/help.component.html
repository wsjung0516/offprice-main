Image 1 Login

  • There are two ways for users to log in: by entering their email and password, or by using social login. If the user has a Google account, they can directly log in by entering their credentials. However, even if Google login is used,
  • If the user wants to log in using the email and password method later, they need to click on "register" once to provide separate email and password inputs.
  • In case of a forgotten password, clicking on "forgot password" will prompt the user to enter their email. After entering the email and clicking the button, they will be directed to a screen where they can enter a new password.
  • After entering the new password on the prompted screen, the user can go back to "https://offprice.store" and log in with the newly set password.

    Image 2 Update User profile

  • If the initial data entered is email and password, the First Name and Last Name fields are empty and require input. If the user logs in via Social Login, the name is already provided.
  • After entering the address, for example, "3416 Manning Ave," wait for a moment. At this point, you can select the City, State, and Country data. Simply choose the desired actual address. The postal code will be automatically populated.
  • Click to select the City.
  • Click to select the State.
  • Click to select the Country.
  • After reading the Terms of use & Privacy policy, click the checkbox to acknowledge acceptance.
  • If the user is a Seller, clicking the checkbox opens a screen to input Seller information. Enter the Store Name, Representative Name, Federal Tax ID (EIN), and Representative Phone Number.
  • Clicking the checkbox will copy the information entered above.
  • Once all the required information is entered, click the "Update" button to complete the submission.

  • The first image selected will become the representative image. If you want to change the representative image, you need to delete all the images and start the selection process again.
