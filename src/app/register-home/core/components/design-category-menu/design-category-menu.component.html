Selected Category Menu

Design Category Menu

  • This feature allows sellers to easily, quickly, and accurately register their products by specifying the categories commonly used by users.
  • Sellers search for the categories they commonly use from the existing registered categories.
  • Click on the found category, and it will be displayed in the "Selected Category Menu".
  • If multiple items need to be selected, repeat the above step (Step 3) for each item.
  • To remove a selected item from the list, click the "close button" of the selected category, and it will be removed from the selected list.
  • After selecting all the relevant categories, click the "Save Category Menu" button.
  • To verify if the saved result has been properly reflected, click the "Register" button in the top menu.
  • Press the Refresh button on the redirected page to navigate to the product registration screen.
  • Check the menu for Category2 on the product registration screen.
